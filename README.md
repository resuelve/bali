# bali
Library focused on the validation of personal and tax identifiers of clients in different documents by country:
- Mexico(RFC, CURP)
- Colombia(CC, CE, NIT)
- Spain(DNI, NIE)
- Portugal(NIF)
- Italy(NIF)
